# Claude Code 全局配置说明

## 语言

- 默认使用 **简体中文**
- 代码中的注释和文档使用中文，除非另有要求

## 代码风格

- 编写简洁、可读、模块化的代码，使用有意义的变量和函数命名
- 遵循当前项目已有的代码规范和模式
- 优先选择简单直接的方案，避免过度设计

## 通用规则

- 需求不明确时，先向用户确认再动手
- 优先编辑现有文件，避免不必要地创建新文件
- 不要添加多余的注释或文档，除非用户要求

## 代码修改

- 所有代码修改，需要给出修改点让用户确认后才能编辑代码文件
- 所有库文件都不可修改，如果涉及到库文件修改，需要给出详细的修改原因，并且让用户多次确认

## MD 文件编辑 / 增加 / 修改

- 文件命名采用中文，除非用户指定

## 多 Agent

- 所有任务优先选择多 Agent（至少 10 个）并行执行，保证执行效率

---

> 配置文件路径：`~/.claude/CLAUDE.md`
